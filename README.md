# Share Content Card Component

A responsive Share Content Card built with **HTML**, **SCSS**, and **JavaScript**.

##  Features

* Share content on:

  * Twitter
  * LinkedIn
  * Facebook
  * Instagram
  * Pinterest
  * Telegram
* Copy link to clipboard
* Hover and active states
* Fully responsive design
* Clean and organized project structure

##  Built With

* HTML5
* SCSS (Sass)
* JavaScript (ES6)

##  Project Structure

```text
src
├── assets
│   └── images
├── css
│   ├── style.css
│   └── style.css.map
├── js
│   └── script.js
└── scss
    ├── _variables.scss
    ├── _global.scss
    ├── _responsive.scss
    └── style.scss

index.html
README.md
```

##  Getting Started

1. Clone the repository.
2. Open the project folder.
3. Run Sass:

```bash
sass --watch ./src/scss:./src/css
```

4. Open `index.html` in your browser.
